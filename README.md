# Loop AI Internship Assignment

Ye simple backend service hai jisme `/ingest` endpoint hai jo POST request leta hai aur data store karta hai.

## Kaise chalaye

1. Node.js install karein.
2. Terminal mein `npm install` karein dependencies install karne ke liye.
3. `node index.js` command se server start karein.
4. Server `http://localhost:3000` pe chalega.
5. `/ingest` endpoint ko POST request bhejein data ke saath.

## Deployment

Ye project maine Render.com pe deploy kiya hai. Aap yeh URL test kar sakte hain:

`https://loopaibackend.onrender.com`
